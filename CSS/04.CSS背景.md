# CSS背景
CSS背景属性用于定义HTML元素的背景。  

CSS属性定义背景效果：  

- background-color
- background-image
- background-repeat
- background-attachment
- background-position

## 背景颜色
background-color属性定义了元素的背景颜色

#### 实例：
```css
body{background-color:#000000;}
```
CSS中，颜色值通常以下方式定义：
- 十六进制："#ff00000"
- RGB："RGB(255,0,0)"
- 颜色名称："red"
